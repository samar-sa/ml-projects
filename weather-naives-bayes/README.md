# Weather Rain Prediction Using Naive Bayes

## Overview
Predicting rainfall based on weather features using Gaussian Naive Bayes.

## Dataset
Weather forecast dataset — features: Temperature, Humidity, Wind Speed, Cloud Cover, Pressure.

## What This Project Covers
- Data exploration and preprocessing
- Label encoding
- Comparing 80/20 and 75/25 train-test splits
- Accuracy score and confusion matrix

## Libraries
- Python, Pandas, Scikit-learn

## Results
- Accuracy (80/20): 97.2%
- Accuracy (75/25): 97.2%

## How to Run
Open in Google Colab and upload weather_forecast_data.csv to session storage before running.
