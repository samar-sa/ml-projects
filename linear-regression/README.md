# Linear Regression with Synthetic Data

## Overview
Implementation of Linear Regression from scratch using NumPy, with and without noise.

## What This Project Covers
- Generating synthetic data
- Computing optimal alpha and beta manually
- Calculating SSE and R2 score
- Comparing results with and without noise

## Libraries
- Python, NumPy, Matplotlib

## Results
- R2 without noise: 1.0
- R2 with noise: 0.7380568226993244
