# Iris Classification Using KNN

## Overview
Classification of Iris flowers into 3 species using K-Nearest Neighbors algorithm.

## Dataset
Iris dataset — 150 samples, 4 features, 3 classes.

## What This Project Covers
- Data exploration and visualization (Pairplot, 3D plot)
- Label encoding
- KNN model training and evaluation
- Cross-validation to find optimal K value
- Confusion matrix and accuracy score

## Libraries
- Python, Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Results
- Best K: (13)
- Accuracy: (97.78 %)

## How to Run
Open in Google Colab and upload Iris.csv to session storage before running.
